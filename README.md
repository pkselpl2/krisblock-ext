# StrongBlock Ext
엔트리에서 여러 비공식 블록을 사용할 수 있도록 해주는 확장 프로그램입니다.

**이 확장 프로그램에 포함된 비공식 블록은 제3자가 개발했으며 예기치 않은 동작이 발생할 수 있습니다.**

## 설치 방법

### 1️⃣ 이 저장소 다운로드
- `Code` → `Download ZIP`
- 또는 `git clone`
- `build` 폴더만 압축 해제

### 2️⃣ 크롬 확장 프로그램 페이지 열기
chrome://extensions

### 3️⃣ 개발자 모드 활성화
- 우측 상단 **개발자 모드 ON**

### 4️⃣ 확장 프로그램 로드
- **압축해제된 확장 프로그램 로드**
- 다운로드한 `build` 폴더 선택

✅ 끝!  
이제 엔트리 작품을 새로고침하면 자동으로 적용됩니다.
